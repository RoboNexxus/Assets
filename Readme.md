# RoboNexxus Assets

A simple asset host for the RoboNexxus club — primarily used to store and serve images (especially `.gif`s) that are embedded in HTML emails and other club materials.

## ✅ What this repo is for
- Hosting GIFs and images for club communications
- Using direct file URLs inside HTML emails
- Keeping commonly used assets in one place

## 📌 How to use
1. Upload your asset into this repository (e.g., `gifs/`, `logos/`, or `banners/`).
2. Open the file on GitHub and copy the **raw file URL**.
3. Use that raw URL in your HTML email or webpage.

Example in HTML:

```html
<img src="https://raw.githubusercontent.com/RoboNexxus/Assets/main/gifs/--__--.gif" alt="--__--">
```

/headers
